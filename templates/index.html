Welcome to Activity Hub!



1. Checking Announcements

  • Go to the Announcements section to find updates on homework, upcoming events, and important reminders.
  • If you’re an admin, you can add or edit these announcements in the Admin Dashboard.


2. Managing Attendance

  • Visit the Attendance section to see who’s been present or absent.
  • Admins can add new students and update attendance records from the Admin Dashboard.


3. Viewing the Schedule

  • The Schedule section shows the CCA timetable.
  • Admins can add or change schedule details using the Admin Dashboard.


4. Admin Features

  • Login: Click on Admin to log in with your admin password and access special features.
  • Add Announcements: Post new announcements about homework, events, or reminders.
  • Update Attendance: Add new students and mark attendance for different days.
  • Manage Schedule: Add or adjust CCA times and sessions.


5. Logging Out

  • To log out, click the Logout link in the admin section.


6. Helpful Tips

  • Keep your admin password safe and secure.
  • Regularly update announcements and attendance for accuracy.
  • Use the schedule to keep track of all your CCA timings.
